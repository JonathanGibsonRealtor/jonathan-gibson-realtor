JONATHAN GIBSON REALTOR WEBSITE

1. Put these files in one folder:
   - index.html
   - style.css
   - vercel.json

2. Create a new GitHub repository named:
   jonathan-gibson-realtor

3. Upload these files to that repo.

4. Go to vercel.com
   - Sign in with GitHub
   - Click "Add New Project"
   - Import the repo
   - Click Deploy

5. After deployment, in Vercel go to:
   Project > Settings > Domains

6. Add:
   - jonathangibsonrealtor.com
   - www.jonathangibsonrealtor.com

7. In Network Solutions DNS, point:
   - A record for @ to 76.76.21.21
   - CNAME for www to cname.vercel-dns.com

8. Wait for domain verification.
